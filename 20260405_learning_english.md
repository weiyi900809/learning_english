###  英文聽力文本：人物訪談 / 商業與人際關係探討

> "What's the secret to building any business relationship, any personal relationship? 
How do you approach that? *(Authenticity)*. 
I think you can't build real relationships if you don't *(genuinely)* like the people you're trying to build them with."


###  重點單字與實用片語解析

* **Authenticity (n.):** 真實性、真誠、不做作。
  > *解析：* 在現代商業與領導力語境中，這是一個極高頻的「大字」。它強調展現真實的自我，而非戴上面具迎合他人。其形容詞為 **authentic** (真實的、道地的)。
* **genuinely (adv.):** 真誠地、由衷地。
  > *解析：* 用來修飾動詞 `like`，強調這種喜歡不是出於利益算計的表面客套，而是發自內心的欣賞。
* **approach (v.):** 著手處理、應付、探討（問題或任務）。
  > *解析：* "How do you approach that?" 在口語中常翻譯為「你是怎麼處理/看待這件事的？」

###  重點文法筆記：不定詞與子句/介係詞的結構差異

* **To 原V (不定詞)：** 可以是 N (名詞)、Adj (形容詞)、Adv (副詞)，但**並非動詞**。
* **子句結構：** 必須由「主詞 (動作著 N) + 動詞 (V)」組成。
  > *例句：You (S) work (V) hard so that (conj) you (S) can get (V) good grades in TOEIC.*
* **介係詞 (prep)：** prep + N，之後不能再有動詞！
* **連接詞 (conj)：** conj + N + V



### 23. 文法題 – 介係詞

All **seats** (座位) must be **confirmed** (確認) **beforehand** (預先) ------ **reservations** (預約) to be **complete** (完整).

- (A) in order to `ph. (片語) 為了... 接原V`
- (B) so that `conj 好讓、為了... 接 S+V`
- **(C) for `conj / prep 為了... 接 S+V 或接 N`** *(註：圖片中此選項有紅框標示)*
- (D) so as `ph. (片語) 為了... 接 to + 原V`



### 題目嚴謹分析

這是一道非常漂亮且容易讓考生掉入陷阱的進階結構題。解題的核心在於圖片最上方的那句綠色筆記：「**To 原V (不定詞)... 並非動詞**」。

**1. 掃描空格後方的結構 (解題最關鍵步驟)：**
* 空格後方的字眼是：`reservations to be complete.`
* `reservations` 是名詞 (N)。
* 後面的 `to be` 是一個「不定詞」，根據文法鐵則，**不定詞絕對不能當作句子的主要動詞 (V)**。
* 因此，`reservations to be complete` 這個區塊**「只有名詞，沒有動詞」**。它不是一個完整的子句 (S+V)。

**2. 逐一檢視選項的文法要求：**
既然空格後面沒有真正的動詞，我們就可以利用刪去法嚴謹排除：
* **(A) in order to：** 後面必須直接接「原形動詞 (V)」。但這裡空格後是名詞 `reservations`，淘汰。
* **(B) so that：** 這是連接詞，後面必須接「完整的子句 (S + V)」。因為後面缺乏真正的動詞，淘汰。
* **(D) so as：** 完整用法是 `so as to + 原V`。同樣不能直接接名詞，淘汰。

**3. 敲定正確答案與句型解析：**
* 唯一在文法上行得通的是 **(C) for**。
* 在這裡，`for` 作為**介係詞**使用，後面接名詞 `reservations`。
* **核心句型補充：** 英文中有一個極為常見的固定句型：**`for + 某人/某物 + to V`**（為了讓某人/某物去執行某動作）。這個 `for` 帶出的名詞，就是後面那個不定詞動作的「邏輯主詞」。
* **語意套入：** 整句話的意思為「所有的座位都必須預先確認，**為了讓 (for)** 預約程序**能夠 (to be)** 完整。」邏輯與文法結構完美吻合。
