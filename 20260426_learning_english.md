###  英文聽力文本：家庭喜劇 / 拙劣的要錢藉口

> "Hey, mom. Yeah? Can I have $40 for lunch? $40? 
I also need a book for school. 
What book? 
I want a dress. 
Do you have any idea what a bad *(liar)* you are? 
I'd be more worried that she couldn't *(come up with)* a single book title."



###  重點單字與片語解析

* **liar (n.):** 說謊者、騙子。
  > *解析：* "what a bad liar you are" 意思是「你這個謊扯得有多爛」。
* **come up with (phr. v.):** 想出、提出（主意、計畫、藉口或答案）。
  > *解析：* 這是一個極度高頻的實用片語。這裡指妹妹竟然連一個書名都「想不出來 / 編不出來」。

### 💡 重點文法與句型筆記

**1. 主動與被動語態**
* **主動：** `S + 主動V + N` (例：I **love** you.)
* **被動：** `S + 被動V + by + N` (例：You **are loved** by Chris)

**2. 假設語氣 (If 條件句) 公式表**

| If 子句結構 (`If + S + ______`) | 主要子句結構 (`S + ______`) |
| :--- | :--- |
| **V(s)** | **will 原V / V(s)** |
| **Ved** (若是 "be" 動詞就選 were) | **Would / could / should / might 原V** |
| **Had PP** | **Would / could / should / might have PP** |

### 📝 44. 文法題 - 假設語氣

If we ------ a **higher** (更高的) **bid** (標價), our company **would have gotten** (取得) the **construction** (建築) **contract** (合約).

- (A) offer
- (B) offered (提供)
- **(C) had offered (主動語態)** *(註：圖片中此選項有紅框標示正確)*
- (D) had been offered (被動語態)


###  題目嚴謹分析與解題邏輯

這是一道結合了**「假設語氣 (Conditional Sentences)」**與**「主被動語態 (Voice)」**的複合型文法題。只要透過圖片上方的兩個筆記區塊，就能用純邏輯的方式精準推導出答案。

**步驟一：透過「主要子句」反推「If 子句」的時態（鎖定假設語氣公式）**
1. 觀察句子後半段的主要子句：`...our company would have gotten...`。
2. 這裡使用了 `would + have + PP (gotten)` 的結構。
3. 對照圖片右上方的「假設語氣公式表」第三列，當主要子句是 `Would / could / should / might have PP` 時，前面的 If 子句必須搭配 **`Had PP`** （這在文法上稱為「與過去事實相反」的假設）。
4. 依據此規則，直接淘汰 (A) offer 與 (B) offered。答案剩下 (C) 與 (D)。

**步驟二：透過「主詞與受詞」關係判斷語態（主動 vs. 被動）**
1. 現在我們要在 (C) had offered（主動語態）和 (D) had been offered（被動語態）之間做選擇。
2. 觀察空格前的主詞是 `we` (我們)，空格後的受詞是 `a higher bid` (一個更高的標價)。
3. 根據圖片左上方的語態筆記 `S + 主動V + N`，邏輯上是「我們（主動）提出更高的標價」，而不是「我們被提出」。因此必須使用**主動語態**。

**結論：**
綜合「假設語氣公式 (Had PP)」與「主動語態邏輯」，**(C) had offered** 是唯一嚴謹且正確的答案。
整句的翻譯為：「如果我們當時**提出了**更高的標價，我們的公司就會取得那份建築合約了。」（暗示：當時並沒有提出更高的標價，所以沒拿到合約）。
